## 二、逐因子理论对照

### 2.1 book_to_price（账面市值比） — Alpha 层

| 维度 | 内容 |
|------|------|
| **书中理论支持** | Fama-French 三因子模型中 HML（High Minus Low）是核心价值因子，学术上被广泛验证。Graham 笔记 L180："低于净流动资产价值的股票…1957年研究：85家此类公司中两年后75%赚钱"。strategy_hypotheses.md A1 明确将价值因子列为 A 股适用性强的策略（✅）。 |
| **与 Ilmanen 的关系** | Ilmanen 1995 论文虽聚焦债券市场，但其核心发现——"预期超额收益与经济周期阶段逆相关"——同样适用于价值因子：价值股在衰退期被低估，在经济恢复时超额收益最高。这与 Fama-French 的"价值溢价是风险补偿"假说一致。 |
| **实际表现** | IR=0.14，在所有因子中最低。Rank IC 约为 0.14/√252 ≈ 0.009，低于 APM 笔记中建议的优秀单因子阈值（IC=0.03-0.05）。 |
| **一致程度** | ⚠️ **方向一致，强度不足**。价值因子的学术逻辑无争议，但在 A 股近年的实际 IC 偏低，可能原因：①注册制后壳价值下降削弱 PB 因子有效性（strategy_hypotheses.md 已预警）；②book_to_price 本身不如 ebit_to_mv/ocf_to_ev 的信息含量高（后者 IR 均为 0.33）。 |
| **改进建议** | 考虑降低 Alpha 层中 book_to_price 的 IR 权重（当前自动使用 IR=0.14），或替换为 BP 的 5 年分位数 Z-Score 版本。 |

---

### 2.2 ebit_to_mv（EBIT/市值） — Alpha 层

| 维度 | 内容 |
|------|------|
| **书中理论支持** | EBIT/EV 是 Fama-French 五因子模型中 profitability 因子的核心变量之一。EBIT 相比净利润排除了利息和税收干扰，更能反映经营效率。这与 strategy_hypotheses.md A6 盈利质量因子（ROE + 盈利稳定性）的理论逻辑一致。 |
| **APM 理论映射** | Grinold 的 alpha = IC × σ_ε × Score 框架中，ebit_to_mv 提供的是 Score 层面的信息。IR=0.33 对应 IC ≈ 0.33/√252 ≈ 0.021，处于 A 股有效因子区间（APM 笔记 L700 列出的有效因子 IC 普遍在 0.02-0.05 之间）。 |
| **实际表现** | IR=0.33，在 Alpha 层并列最高。作为基本面因子，信号换手率低（季报频率更新），与 APM 对"广度"（Breadth）的要求兼容。 |
| **一致程度** | ✅ **高度一致**。盈利能力因子的学术和实证证据充分，A 股表现与理论预期吻合。 |
| **改进建议** | 可考虑加入 EBIT 的同比变化率（盈利动量），APM 笔记提到分析师预期修正因子是 A 股有效因子之一。 |

---

### 2.3 ocf_to_ev（经营现金流/企业价值） — Alpha 层

| 维度 | 内容 |
|------|------|
| **书中理论支持** | 现金流因子是盈利质量因子的延伸。Graham 笔记 L460 警告"投资者的主要损失来自有利的商业条件下购买劣质股…购买者往往将现在的好收益等同于收益能力"。OCF/EV 正是区分"真实盈利"与"纸面盈利"的关键指标——经营现金流难以通过会计操纵。strategy_hypotheses.md A6 提到需加入财务排雷指标（应收账款增速>营收增速、经营现金流/净利润<0.5 等）。 |
| **与价值因子的互补** | ocf_to_ev 同时包含价值信息（EV 分母）和质量信息（OCF 分子），是比单纯 BP 更全面的因子。这解释了其 IR（0.33）远高于 book_to_price（0.14）。 |
| **实际表现** | IR=0.33，与 ebit_to_mv 并列 Alpha 层最高。 |
| **一致程度** | ✅ **高度一致**。现金流因子在学术上被认为比会计利润更难操纵，更接近"真实价值"。A 股财务造假风险高（strategy_hypotheses.md 已指出），OCF 因子尤为关键。 |

### 2.4 vol_std_20d（20日收益波动率） — Risk 层

| 维度 | 内容 |
|------|------|
| **书中理论支持** | **低波动率异象**：APM 笔记 L699-700 明确将"低波动率异常"列为 A 股有效因子之一。strategy_hypotheses.md A4 详细讨论了低波动率因子：与 CAPM 矛盾，但实证稳健。Carver 笔记 L36-41 的波动率目标框架（Volatility Targeting）间接支撑低波动率因子：通过降低高波动资产仓位、增加低波动资产仓位，本质上在做低波动率倾斜。|
| **Carver 的仓位管理** | Carver 的核心公式：目标仓位 = σ_target / σ_current。当 σ_current 高时减仓，低时加仓。Baseline 中 vol_std_20d 使用 negate=True（越低越好），与 Carver 的逻辑完全一致——低波动资产获得更高评分。 |
| **实际表现** | IR=0.31，在 Risk 层中低于 turnover_rate_f（0.42）但仍然显著。使用 20 日窗口而非 60 日，对波动率变化的反应更灵敏，但也更容易受到短期噪声影响。 |
| **一致程度** | ✅ **高度一致**。低波动率异象在 APM 和 Carver 两本书中都有理论支撑，A 股散户追涨杀跌导致高波动股被系统性高估。 |
| **潜在冲突** | **窗口长度选择**：strategy_hypotheses.md A4 建议检验 20/60/120 日窗口。当前 20 日窗口偏短，可能与 APM 笔记中讨论的"波动率因子的半衰期"不完全匹配。APM 提到低波动率因子的信号半衰期约 1-3 个月，60 日窗口可能更优。 |

### 2.5 turnover_rate_f（自由流通换手率） — Risk 层

| 维度 | 内容 |
|------|------|
| **书中理论支持** | APM 笔记 L698："有效的 A 股因子：低换手率组合表现优异"。strategy_hypotheses.md A5 专门讨论低换手率因子：高换手率反映市场过度关注和投机，股价已被充分交易甚至过度交易。A 股整体换手率远高于成熟市场，高换手率股票往往已被充分炒作。 |
| **Carver 的视角** | Carver 的框架更关注波动率而非换手率，但二者高度相关：高换手率 → 高交易成本 → 净收益降低。Carver 在成本控制上的强调间接支持低换手率因子。 |
| **实际表现** | IR=0.42，是所有 10 个因子中**最高**的。这验证了 APM 的判断——低换手率在 A 股确实是最强因子之一。 |
| **一致程度** | ✅ **高度一致，IR 最高验证理论判断**。A 股散户主导的市场结构中，过度交易是系统性噪音源，低换手率因子捕获的是"被忽视的价值"。 |

### 2.6 mom_20d（20日动量） — Enhance 层

| 维度 | 内容 |
|------|------|
| **书中理论支持** | **动量因子的时间维度**：strategy_hypotheses.md A2（12-1 动量）引用 Chan 的实证数据，截面动量在正常市况下 APR 37%、Sharpe 4.1，但在危机中 APR -30% 严重失效。APM 笔记讨论 alpha 信号的衰减：短期动量（1-3 个月）的半衰期较短，但信号强度更高。 |
| **与 APM 的一致性** | Grinold 的 IR = IC × √BR 框架中，动量因子的高换手率（月频调仓）带来更大的 BR（Breadth），部分补偿了 IC 的衰减。mom_20d 使用 20 日窗口而非 Chan 的 252 日，属于"短期动量"范畴，与 APM 笔记中 A 股有效因子"短期反转（5-20日）"有部分重叠。 |
| **与反转因子的关系** | strategy_hypotheses.md A3（5-20日短期反转）与 mom_20d 存在理论张力：反转认为过去跌的会涨回来，动量认为过去涨的会继续涨。20 日窗口恰好处于反转和动量的"灰色地带"。 |
| **实际表现** | IR=0.38，在 Enhance 层最高。 |
| **一致程度** | ⚠️ **方向一致但有理论张力**。20 日动量与 5-20 日反转因子存在信号方向冲突。实际 IR=0.38 说明在 A 股中短期动量占优，但需注意动量崩溃风险（strategy_hypotheses.md 矛盾对照表明确指出）。 |
| **改进建议** | 建议在 Enhance 层加入 mom_20d 与反转因子的条件切换逻辑：市场状态判断为"趋势市"时使用动量，"震荡市"时切换为反转。 |

### 2.7 bbi_momentum（BBI 动量） — Enhance 层

| 维度 | 内容 |
|------|------|

## 三、组合构建方法的理论对照

### 3.1 IR 加权因子组合

| 维度 | 内容 |
|------|------|
| **书中理论支持** | **Grinold 的信息比率框架是 IR 加权的直接理论来源**。APM 笔记核心公式：**IR = IC × √BR**，组合层面 IR = √(Σ wᵢ² IRᵢ² + 2Σᵢ<ⱼ wᵢwⱼρᵢⱼIRᵢIRⱼ)。当因子间不相关时，最优权重 ∝ IRᵢ。代码实现 `compute_layer_score()` 中 `weights = abs(ir_weights.get(c, 1.0))` 正是按 \|IR\| 加权。 |
| **实际实现** | 代码中三层分别做 IR 加权（层内），再用固定权重 0.55/0.20/0.25 做层间加权。层内 IR 加权 ≈ Grinold 最优解（假设因子间低相关）；层间固定权重 ≠ 理论最优。 |
| **一致程度** | ✅ **层内高度一致，层间可优化**。Grinold 理论的最优层间权重应 ∝ IR × √N（N 为该层因子数量），而非固定 0.55/0.20/0.25。 |
| **改进建议** | 层间权重可用 Alpha/Risk/Enhance 各层的组合 IR 动态计算：w_cat ∝ IR_cat × √N_cat。 |

---

### 3.2 行业中性化 Cross-Sectional Rank

| 维度 | 内容 |
|------|------|
| **书中理论支持** | **Grinold 的纯因子组合（Pure Factor Portfolio）理论**要求因子暴露与行业暴露正交化。APM 笔记中讨论的"行业中性化处理（申万一级行业）"正是此意。代码 `neutralize_by_industry()` 在 rank 之前对每个因子按行业-日期去均值，确保因子得分不反映行业偏差。 |
| **Carver 的补充** | Carver 笔记 L92-95 的等波动率权重框架也隐含行业分散——不同行业的波动率差异大，等波动率权重会自然降低高波动行业（如半导体）的权重。 |
| **实现细节** | 代码中先做 `neutralize_by_industry()`（去均值），再做 `groupby(level="datetime").rank(pct=True)`（截面百分位排名），顺序正确——先去除行业偏差，再做排名标准化。 |
| **一致程度** | ✅ **高度一致**。两步处理（行业去均值 + 截面 rank）完全符合 Grinold 纯因子组合的标准做法。 |
| **潜在问题** | 仅做行业中性化未做**市值中性化**。APM 笔记提到 A 股低波动率因子可能被市值偏差污染（低波动股多为银行等大盘股）。建议在行业中性化基础上加入市值中性化（或至少做 size 分层 IC 检验）。 |

## 四、Lopez de Prado 的 ML 视角审视

虽然 Baseline 目前不使用 LightGBM/ML 模型（代码中无 lgb/xgb 引用），但 LdP 的框架对因子组合仍有重要启示：

| LdP 原则 | Baseline 现状 | 差距 |
|-----------|--------------|------|
| **Feature Importance 分析**（Ch.8） | IR 值充当了简单的 feature importance | ✅ 基本满足，但 IR 是单因子指标，未考虑因子间交互效应 |
| **Purged K-Fold CV**（Ch.7,9） | 未见 Purged CV 实现 | ❌ 回测可能存在信息泄露（因子计算中 qlib source 因子使用滚动窗口，parquet 因子依赖财报发布时间） |
| **Triple Barrier 标注**（Ch.3） | 使用固定窗口收益率作为标签 | ⚠️ 未考虑波动率差异对标签质量的影响 |
| **过拟合检测**（Ch.11-15） | 未见 Deflated Sharpe Ratio 或 Combinatorial Purged CV | ❌ 缺乏过拟合概率的定量评估 |
| **Meta-Labeling**（Ch.3） | 无 | ⚠️ 可用于将因子信号转为概率，提升组合的鲁棒性 |

## 五、关键发现汇总

### 理论高度一致之处 ✅
1. **价值/盈利因子三件套**（ebit_to_mv, ocf_to_ev, roa）：与 Fama-French 五因子 + APM A 股有效因子完全一致
2. **低波动率 + 低换手率**（vol_std_20d, turnover_rate_f）：与 APM 低波动异象 + Carver 波动率目标框架一致
3. **IR 加权 + 行业中性化**：完全符合 Grinold 的信息比率框架和纯因子组合理论
4. **低换手率因子 IR=0.42 最高**：验证了 APM 对 A 股因子有效性的判断

### 存在张力或可改进之处 ⚠️
1. **book_to_price IR=0.14 最低**：注册制后壳价值下降，PB 因子需要升级（如 5 年分位数 Z-Score）
2. **mom_20d 与反转因子的灰色地带**：20 日窗口在动量和反转之间不清晰，需条件切换
3. **bbi_momentum 可能与 mom_20d 冗余**：两个价格趋势因子 IR 差异大（0.26 vs 0.38），需检验相关性
4. **层间权重固定而非 IR 最优**：0.55/0.20/0.25 是经验值，未按 Grinold 理论动态优化
5. **缺少市值中性化**：仅行业中性化不够，低波动因子可能隐含市值偏差
6. **波动率窗口偏短**：vol_std_20d 使用 20 日窗口，APM 建议的信号半衰期 1-3 个月对应 20-60 日

### 理论冲突之处 ❌
1. **动量 vs 反转的根本矛盾**（strategy_hypotheses.md 矛盾对照表）：Chan 实证动量 APR 37% 但危机中 -30%；APM 说 A 股短期反转（5-20日）效果显著。20 日 mom 和 5-20 日反转在相同窗口上给出相反信号。
2. **分散化 vs 集中化**（矛盾对照表）：Carver 的等波动率权重追求分散，但 Graham/Lynch 倾向集中持仓。Baseline 的 IR 加权自然倾向集中（高 IR 因子权重大），与 Carver 的等风险贡献哲学不完全一致。
